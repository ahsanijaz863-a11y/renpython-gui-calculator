# 🧮 Python GUI Calculator

A simple and interactive calculator built using Python and Tkinter.  
This application performs basic arithmetic operations with a clean graphical user interface.

---

## 🚀 Features
- ➕ Addition  
- ➖ Subtraction  
- ✖️ Multiplication  
- ➗ Division  
- 🖥️ GUI-based input using buttons  
- ❌ Error handling for invalid expressions  
- 🔄 Clear button to reset input  

---

## 🛠️ Technologies Used
- Python  
- Tkinter  

---

## ▶️ How to Run

```bash
python calculator.py
